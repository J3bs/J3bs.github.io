# J3IT Branding Standards

This repository is for developing and iterating on branding standards for J3IT, including logo concepts, color palettes, typography direction, and usage guidance.

## Current Contents

- `assets/logo-proofs/` - first-pass SVG logo proof concepts
- `proofs.html` - local proof board for manual review
- `focused-proofs.html` - second-stage proof board for options 3 and 5
- `option-3-proofs.html` - separated option 3 proof board
- `option-5-colourOptions.html` - option 5A-E color palette matrix
- `option-5-caseOptions.html` - option 5A-E wordmark case matrix
- `docs/brand-direction.md` - working brand direction and review notes
- `docs/focused-proof-philosophy.md` - visual rules used for focused variants

## Review Workflow

Open `proofs.html` in a browser, choose the strongest directions, and note:

- concept number
- what works
- what should change
- where the logo needs to be used

The next iteration should refine selected concepts rather than keep expanding the set.
